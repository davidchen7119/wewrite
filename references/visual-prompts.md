# 视觉 AI 模块 — 中老年人生哲理文案配图规范

> 本文件定义 WeWrite 的配图提示词生成规则。**最高级硬性约束**：所有提示词用**中文**输出；图片必须描绘中老年女性的**真实生活场景瞬间**；金句卡片图的**金句文字必须出现在图片中**；**视觉风格必须锁定，绝不偏移**（见下方「风格锁」）。

## 〇、风格锁（STYLE LOCK · 全程统一，绝不偏移）

> ⚠️ 最高级红线：以下风格描述是一套**单一、连续、出版级绘本系列**的定义。每一张提示词（封面 + 每张卡片）都必须**完整嵌入这段风格锁**，绝不允许简化、混搭其他画风、或逐张微调导致系列不统一。

**风格锁·基础段（氛围图 + 金句卡片图共用，逐字拷贝）：**

> 日式极简自然主义治愈系绘本插画，几米风格，小森林美学，东方留白美学，生活纪实绘本风格。画面采用极简构图，大面积留白，突出主体，强调视觉呼吸感，以富有电影感的镜头语言讲述平凡生活中的情绪瞬间。人物比例自然略带Q版，五官简洁克制，动作真实生动，具有生活观察感，不刻意夸张，不刻意卖萌，通过姿态、细节和环境传递情绪。采用彩色铅笔、水彩与粉彩混合手绘表现，细腻而略显稚拙的线条勾勒轮廓，轻微不规则手绘感，柔和笔触，自然毛边，无明显描边。保留粗糙和纸、水彩纸纹理及细腻颗粒感，低锐度渲染，仿传统纸本绘画效果，具有真实出版级绘本质感。光影遵循真实环境光逻辑，根据场景自动匹配自然光源与环境氛围，可表现晨光、午后、黄昏、夜晚灯光、室内暖光、窗边侧光、逆光、漫反射等真实照明方式，不使用强烈戏剧化布光，不出现高对比硬阴影，整体光线柔和自然，富有空气感和空间层次。整体采用低饱和、高明度的莫兰迪色彩体系，以暖白、米白、浅木色作为基础色调，根据画面内容自然延展浅蓝、浅绿、浅棕、浅灰、暖橙等柔和色彩，避免高纯度、高对比配色，整体色彩克制、统一、舒适，具有温暖宁静的视觉感受。画面强调生活叙事与情绪表达，不追求摄影级写实，而是通过人物姿态、环境细节、光线、留白和色彩表达故事，让画面安静、细腻、克制、治愈，具有诗意和情感共鸣，充满生活温度。整体保持统一系列绘本风格，统一人物设计语言，统一笔触语言，统一纸张肌理，统一色彩体系，统一光影逻辑，统一镜头语言，统一情绪表达，统一留白构图，保持连续绘本视觉风格，具有高级出版级原创绘本插画质感。

**风格锁·文字附加段（仅金句卡片图使用，拼在基础段之后）：**

> 画面中的文字与插画共同参与构图，根据主体位置、画面节奏和视觉重心自然融入整体版式，使文字成为画面的一部分，而非独立的信息层；文字位置可根据构图自由分布于画面的任意合适位置，与主体、留白、光影和环境元素形成视觉呼应，整体布局自然均衡、富有呼吸感；文字采用娟秀精致的手写字体，笔画细腻流畅，具有东方书写韵味和绘本气质，字形温润克制，保持适度留白与舒展的行距、字距，形成富有设计感的诗意排版；文字与插画在色彩、笔触和纸张肌理上保持统一，仿佛自然书写于画纸之上，与画面融为一体，共同营造高级绘本、诗集插画与文艺海报般的视觉美感。

> **使用规则**：
> - **氛围图（封面）**：只嵌入「风格锁·基础段」，**不嵌入文字附加段**。
> - **金句卡片图**：嵌入「风格锁·基础段 + 文字附加段」（即完整版）。

**锁定要点（自检用）：**
- 画风：日式极简 + 几米 + 小森林 + 东方留白 + 生活纪实绘本。禁止写实摄影、禁止厚涂厚描边、禁止卡通可爱化、禁止高饱和网红风。
- 构图：极简、大留白、主体突出、电影感镜头。
- 人物：自然略带Q版、五官简洁克制、动作真实生动、生活观察感；不夸张不卖萌。
- 媒材：彩铅 + 水彩 + 粉彩混合手绘，稚拙线条、自然毛边、和纸/水彩纸纹理、低锐度。
- 光影：真实环境光（晨光/午后/黄昏/窗边侧光/室内暖光等），柔和、无硬阴影。
- 色彩：低饱和高明度莫兰迪，暖白/米白/浅木为底，延展浅蓝绿棕灰橙；克制统一。
- 文字：仅金句卡图嵌入「文字附加段」，金句用手写体自然融入画面，与插画一体；氛围图不嵌入文字附加段、不含任何文字。

## 一、总原则（不可违背）

1. **提示词全部用中文**——适配即梦/seedream 中文理解，禁止输出英文 prompt。
2. **真实生活场景**——图片描绘的是 45-60 岁女性日常生活中**真实会发生的一瞬间**，不是抽象隐喻、不是摆拍、不是画感过强的"概念图"。
3. **共鸣优先**——中老年女性读者看到图，要能脱口而出"这说的不就是我吗"。
4. **金句文字入图**——金句卡片图的画面中，**必须用清晰可读的手写体把金句文字写出来并自然融入画面**（即梦/seedream 支持中文文字生成）。
5. **风格绝对锁定**——每一张提示词都**完整嵌入「风格锁」原文**，严禁简化、混搭或逐张漂移；全篇是一套连续绘本系列。
6. **图片一律呈现正面、阳光、积极的情绪状态**——即便某段文字点名的是"疲惫/委屈/焦虑"等负面情绪，生成的图片也**必须画"走出痛苦后的轻松与光亮"**：觉醒、松弛、自在、温暖喜悦。这是图片的**基础情绪**，任何画面都不得偏离。可在此基础情绪之上**酌情叠加**"中老年女性找回自我、重拾兴趣"的生活瞬间（如遛狗、插花、拍照、跟姐妹旅游——**仅为举例，绝非清单，须发挥想象力扩展出更多场景，绝不可只在这几个里打转**），但兴趣/事业只是**可选的增强层**，**绝非替代基础情绪**。禁止阴沉、疲惫、哭丧、压抑的画面。读者看图的感受应是"我也想这样"，而非"又被戳中痛点"。
7. **画面主体须与金句贴合（贴合判断 · 不强行硬塞）**——若要在基础情绪上叠加"兴趣/找回自我"的瞬间，该瞬间必须能呼应对应段落的金句内核。生成前先判断：这个生活瞬间，是不是在"演"这句金句想传递的那种自我状态？**合适 → 用作增强层；不合适/牵强 → 坚决不下放兴趣，退回纯"阳光/觉醒/松弛"的情绪画面，绝不为凑热闹硬塞。** 基础情绪永远优先，兴趣只是锦上添花。

## 二、角色一致性（当用户提供了角色参考图时）

1. **全系列统一角色**：封面、金句卡片图必须使用**同一中年女性角色**。
2. **提取角色锚点**：从用户提供的角色参考图读取面部特征、发型、神态、色彩板，形成文字版 `character_anchor.md`，并存入 `{cwd}/WeWrite输出/{slug}/character_anchor.md`。
3. **脸部必须保持一致**：不同场景可更换服装、配饰、姿态，但**面部轮廓、五官比例、发型、神态必须与参考图一致**。
4. **场景适配服装**：根据画面情境给角色换合适的服装（如遛狗时休闲风衣/运动鞋、插花时宽松围裙、窗边独处时柔软毛衣、旅行时轻便外套/帽子等），服装风格须与绘本低饱和、柔软材质、生活化调性一致。
5. **提示词写法**：每条提示词必须写明角色一致性，如"同一中年女性角色，面部特征不变，换穿……"。
6. **工具配合**：在即梦/seedream 中，强烈建议上传角色参考图作为"角色参考/人物一致性/垫图"图片，单靠文字描述无法 100% 锁定脸部。
7. **背影场景不描写面部**：若某张图的设计是主角**背影**（如遛狗时背身走远、窗边望向窗外、独自前行等），提示词中**严禁出现任何面部特征描写**——不写脸型、肤色、眼睛、发型、神态等角色锚点内容，只写"背影""后脑勺""发尾/衣领""身影"等。角色锚点（面部/发型/神态）仅在正面或清晰侧面的图中出现；背影图靠"服装/身形/发尾"延续角色辨识度即可。
8. **配角不写五官、以主角为主**：当场景中除主角外还出现其他人（如姐妹游的同伴、路人、店员等），提示词中**一律不写配角的五官与表情**；无论配角在图中处于什么姿态（侧面/背面/正面/远景等，按画面实际决定，不作强制规定），都须"以主角为主"——配角不抢占画面中心、不喧宾夺主，仅作为环境氛围的点缀。主角永远是视觉焦点。

## 三、生成流程（必须按顺序执行）

### 第一步：吃透三段文稿核心

读取终稿的 #01 / #02 / #03，逐段提炼：

| 段落 | 要提炼的内容 |
|------|------|
| #01 | 抛出的核心观点是什么？读者此刻的**痛点情绪**是什么？（心疼/委屈/觉醒） |
| #02 | 展开的生活矛盾是什么？读者此刻的**痛点情绪**是什么？（疲惫/醒悟） |
| #03 | 升华落点是什么？读者看完应有的**释然情绪**是什么？（松弛/自在） |

**⚠️ 情绪锚点双層解读（最高级重点）**：每一段要提炼出**两层**情绪，二者绝不可混淆：
- **第一层 · 文中读者情绪（痛点）**：文字点名的、读者正在经历的负面感受（如 #02 的"疲惫"）。
- **第二层 · 图片应呈现情绪（正面阳光·找回自我）**：文章想带读者**抵达**的状态——觉醒、松弛、自在、温暖喜悦，且最好落在"她正在做一件属于自己的事/兴趣"上。

**生成的图片必须呈现第二层（正面阳光 + 找回自我的积极状态），而不是重复第一层（痛点）**。即：#02 文中是"疲惫"，画面就要画"觉醒后、终于为自己做点什么（如独自拍照/插花）的松弛与轻盈"；#01 文中是"为别人活"，画面就要画"开始好好爱自己、重拾兴趣的从容喜悦"。

### 第二步：判断是否需要叠加「兴趣/找回自我」瞬间（可选增强层）

基础情绪（阳光/觉醒/松弛）已是合格画面。此步仅判断是否**锦上添花**地叠加一个"她在做自己喜欢的事"的瞬间：

- 先确认基础情绪画面（如窗边松弛独处、迎光浅笑）已成立。
- **发挥想象力，从下方（远不穷尽的）素材库挑一个真正贴合金句内核的瞬间作为增强层**。⚠️ **遛狗/插花/拍照/跟姐妹旅游只是用户随口举例，绝不可只在这四个里打转**——同一句金句可匹配十几种场景，优先选**最贴金句内核、最新颖、最有生活质感**的真实瞬间：
  > **独处·自我宠爱**：阳台晒太阳发呆 · 泡茶读书 · 泡澡点香薰 · 写日记/手账 · 听老唱片 · 撸猫 · 对镜涂口红浅笑 · 给自己做一顿喜欢的饭 · 雨天窝沙发织围巾
  > **园艺·手作·慢生活**：养花浇水 · 多肉小王国 · 插花 · 烘焙蛋糕 · 钩针编织 · 做陶艺 · 扎染 · 茶艺 · 香道 · 临帖写字
  > **学习·创造**：学画画 · 练书法 · 学琴 · 学篆刻 · 学外语 · 用手机拍视频记录生活 · 写作/博客 · 集邮 · 做手工
  > **运动·户外·健康**：晨跑/夜跑 · 健步走/遛狗 · 骑行 · 爬山徒步 · 游泳 · 打太极 · 八段锦 · 钓鱼 · 露营 · 逛早市挑菜 · 逛花市
  > **联结·绽放**：闺蜜下午茶 · 老同学聚会 · 读书会/合唱团 · 社区活动 · 做公益志愿者 · 跟姐妹旅行 · 广场舞队伍 · 交谊舞
- **贴合判断（最高级）**：这个兴趣瞬间，是否在"演"该句金句想传递的自我状态？
  - 合适 → 用，作为增强层叠加在基础情绪上。
  - 不合适/牵强 → **不下放兴趣**，保留纯基础情绪画面，绝不为凑热闹硬塞。
- **贴合判断参考（非固定答案，场景可任意替换）**：
  - 金句讲"快乐是你自己的" → 她独自享受某兴趣（插花/泡茶/织围巾/临帖）很贴，可用。
  - 金句讲"别总把别人放前头" → 基础"觉醒松弛"已够；硬加兴趣易牵强，优先保留纯情绪画面，或仅用极轻的"为自己留白"瞬间（如阳台发呆、对镜涂口红）。
  - 金句讲"爱自己/释然" → 与姐妹同游、闺蜜下午茶、或自在独处皆贴，可用。

### 第 2.5 步：温言入画图片去重与景别规划

在动手写提示词之前，先完成两件事，避免新稿与已发布图撞场景+动作：

1. **读取库存**：读取 `{skill_dir}/references/visual-scene-inventory.md`，并尝试用 IMA 工具拉取「温言入画」库最新图片，建立已占用「场景 + 人物动作」清单。
2. **比对去重**：列出本次 4 张图（1 氛围图 + 3 金句卡片图）的 planned 场景 + 动作，与库存比对：
   - 若「场景 + 动作」与已发布图相同，则必须在**服装、场景、道具、人物姿态、景别**中**至少 2 项**明显不同，否则重写该图。
   - 若场景相同但动作不同（如已发布"窗边喝茶"，新图"窗边远眺"），通常可算 1 项差异，还需再改服装或道具 1 项。
3. **景别规划**：为 4 张图分配景别，**仅允许全景 / 中全景 / 中景**，且必须**有所区分**（不得 4 张同一景别）。每张提示词显式写明"全景构图 / 中全景构图 / 中景构图"。

### 第三步：全网搜索真实生活场景（CRITICAL）

用 `WebSearch` 工具，针对选定瞬间搜索**中老年女性真实生活场景**关键词（如"中年女性 阳台 插花 日常""中年女性 清晨 遛狗 小区"），提取 2-3 个可画面化的具体瞬间作为画面主体。

**硬规则**：画面主体必须来自真实生活场景，不得用"抽象概念""符号化隐喻"替代。自检：没读过文章的人看这张图，能联想到"中年女性的日常"吗？不能 → 重写。

### 第四步：撰写中文提示词

按下方模板生成。每条提示词 = **画面描述（中文）+ 情绪 + 风格锁（完整嵌入）+ 金句文字（仅金句卡图）+ 禁用英文文字乱码说明**。

描绘与全文情绪最贴合的一个"找回自我的积极状态"瞬间，定调但不载论点。**氛围图不含任何文字，提示词只嵌入「风格锁·基础段」，不嵌入文字附加段。**

```
- 情绪基调：{明亮温暖 / 松弛自在，与全文开头情绪对齐}
```

```
- 贴合判断：{一句话说明此瞬间如何呼应金句内核}
- 情绪基调：{与 #N 段第二层情绪一致}
- 中文提示词：

```

**数量规则**：文章有几段【金句卡片】，就生成几张金句卡片图提示词。如 #01/#02/#03 各有金句卡片 → 生成 3 张；如只有 #01/#02 有 → 生成 2 张。**氛围图不计入此数量。**

**金句文字入图自检**：提示词中必须含"用手写体自然融入文字：'金句全文'"这一句。缺此句 → 打回重写。

### 景别控制与去重

1. **仅允许景别**：全景、中全景、中景。4 张图（1 氛围图 + 3 金句卡片图）必须**景别有所区分**，不得 4 张同一种景别。
2. **禁用景别**：特写、近景、远景、大特写、大远景。任何提示词出现禁用景别 → 重写。
3. **提示词写法**：每张提示词必须显式写明"全景构图 / 中全景构图 / 中景构图"，并据此控制画面取景范围。
4. **图片去重**：生成前读取 `{skill_dir}/references/visual-scene-inventory.md` 并尝试拉取 IMA「温言入画」库最新图片，建立已占用"场景 + 人物动作"清单。新图 planned 场景 + 动作与已发布图重复时，必须在**服装、场景、道具、人物姿态、景别**中**至少 2 项**明显不同，否则重写该图。

## 六、辅助功能

- **提示词修改**：用户说"#01 图想要更暖的色调"→ 只改对应提示词，其他不变，但风格锁基础段/文字附加段不动。
- **场景调整**：用户说"这段换遛狗场景"或"换个兴趣"→ 按贴合判断重选，重写对应提示词。
- **金句改字**：金句文字变更 → 同步改提示词中的"融入文字"部分，保持逐字一致。
- **风格守护**：任何情况下不得为单图引入新画风、新笔触、新色系；一旦用户要求"换风格"，须整组统一替换风格锁。
