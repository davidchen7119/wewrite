## 核心用户
- 主体：**45—60 岁女性**（兼顾 35—60）。
- 她们在想：身体状态、精神状态、重新认识自己、后半程人生质量。
- 热点搜索目标：找与她们**真实生活相关**的话题（正在经历 / 担心 / 向往 / 愿分享），而非追新闻热度。

## 八大搜索领域

### 领域一：女性健康与身体转型（关注 35—60）
- **搜索关键词**：女性健康 / 更年期 / 激素变化 / 睡眠问题 / 体重管理 / 血糖 / 骨质疏松 / 运动健康 / 情绪波动 / 抗衰老
- **重点搜索**：身体变化的新研究、专家健康建议、女性健康趋势、大众健康误区
- **内容方向**：不制造年龄焦虑；讲"如何理解身体变化，并更好照顾自己"
- **⚠️ 禁忌**：年龄焦虑、身材羞辱、伪科学养生

### 领域二：职场变化与人生转型（关注 35—55）
- **搜索关键词**：女性职场 / 中年危机 / 职场转型 / 女性创业 / 副业 / 再就业 / 退休规划
- **重点搜索**：中年女性职业变化、女性重新开始案例、40+ 女性创业故事、职场女性成长
- **内容方向**："人生下半场，还有哪些可能"

### 领域三：家庭关系与情感变化（关注 35—60）
- **搜索关键词**：中年婚姻 / 夫妻关系 / 空巢家庭 / 母女关系 / 子女独立 / 婆媳关系 / 家庭边界
- **重点搜索**：子女长大后的亲子关系、夫妻进入中年的相处模式、女性如何摆脱家庭过度消耗
- **内容方向**：从"如何照顾别人"转向"如何照顾自己"

### 领域四：熟龄女性美学与自我表达（关注 40—60）
- **搜索关键词**：熟龄女性 / 中年穿搭 / 女性气质 / 白发美学 / 身材管理 / 优雅老去 / 女性魅力
- **重点搜索**：40+ 女性审美趋势、熟龄女性生活方式、女性自信表达
- **内容方向**：避免逆龄 / 装嫩 / "永远 18 岁"；强调成熟魅力、自信、高质量生活

### 领域五：退休与人生第二阶段（关注 45—60）

- **内容方向**："人生不是走向结束，而是进入新的阶段"

### 领域六：养老准备与未来生活（关注 45—60）
- **搜索关键词**：养老规划 / 居家养老 / 社区养老 / 养老社区 / 适老化 / 独居养老

### 领域七：财富、安全与消费趋势（关注 40—60）
- **搜索关键词**：女性财富 / 家庭资产 / 养老金 / 银发经济 / 消费升级 / 防诈骗

## 检索纪律（红线，务必遵守）
- **不编造**数据或 quote；来源存疑只在"思路"层面借鉴，不当文中事实引用。
- **不制造年龄焦虑、不身材羞辱、不伪科学、不硬广**。
- 素材须能自然嵌入 #02（或 #01/#03）做"引用 / 案例 / 画面"，**不破坏"你"的对话感与金句双发机制**。
