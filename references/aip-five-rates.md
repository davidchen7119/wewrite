# AIP 五率优化（完播 / 评论 / 点赞 / 关注 / 真实度）

> 本文件定义温言入画公众号文稿的 **AIP 五率治法**：在 Step 4 成稿（温言入画文风 + Mode B 真实事件开篇）之上，针对公众号核心指标做定向增强。
> 来源：wechat-aip-architect skill 的「勾人开头 + 互动型/金句型结尾 + 融入 IP」打法，经「别再上赶着」一稿（2026-07-18）验证为全号标准增强层。
> **优先级**：本文件是对成稿的**增强层**，**不覆盖** `writing-guide.md` / SKILL.md 4.x 的铁律（三段 + 金句双发 + 第二人称 + 温暖笃定 + 固定 CTA）。调用时机：Step 4 成稿后、Step 5 自检前。

---

## 一、五率治法（落到成稿的具体动作）

| 指标 | 解法（落到本文） |
|------|------|
| **完播率 / 停留时长** | **开篇用真实事件制造好奇钩子**（Mode B）。开头即冲突/悬念，结尾给落点，形成"开头疑问→结尾答案"的闭环，勾着读者划到底 |
| **评论率低** | 文末加**评论引导提问**，诱发"我妈 / 我身边也有这样的人"共鸣评论（如："你有过……的时候吗？或者，你身边也有个像 XX 这样默默付出的人？评论区跟温言说说"） |
| **点赞率低** | 保留并强化文末「赞👍🏻 + 在看❤️」引导，与「你每一份认可，都是温柔的鼓励」承接 |
| **关注率低** | 文末加**关注钩子**（"关注温言入画——往后每天，都有一句掏心窝的话陪你"）+ #03 温言人设落点，强化 IP 承接与涨粉理由 |
| **真实度 / 反 AI 味** | 真实事件须来自**真实报道**（去营销化、标注来源），不编造证人 / 情节；事件细节以报道原文为准 |

---

## 二、使用纪律

- 五率增强**一律追加在成稿末尾 CTA 区**（评论引导 + 点赞 + 关注钩子），不侵入三段主体说理。
- 关注钩子为涨粉必需，但属文稿层增强；若多篇沿用、想固化进模板，须用户明确同意后再写回 skill（不擅自改 `writing-guide.md`）。
- 不破 wewrite 铁律：`#01/#02/#03` 三段 + 金句双发逐字一致 + 第二人称"你" + 温暖笃定 + 固定 CTA。
- 真实事件来源纪律（不可破）：经 `anysearch` / `WebSearch` 取真实报道 / 论坛一手，剔除营销号、软广、蓝 V；**禁止编造**证人、情节、对话；引用处标注来源（如"据腾讯新闻公开报道"）；检索不到贴合事件则回退"你"的共情画面开篇，不硬凑。
- 优化后字数通常 **800–1200 字**（因融入真实事件展开）；未走本增强层的短文仍 200–400 字。

## 三、已验证落位示例（「为什么你越主动，别人越不珍惜你」）

- **停留**：#01 以"62 岁刘姨被备注'保姆（免费）'→ 当晚收拾行李走，只说'我累了'"真实事件开篇，制造好奇；结尾落回标题问句形成闭环。
- **评论**：文末提问"你有过上赶着去暖一块捂不热的石头的时候吗？或你身边也有像刘姨这样默默付出的人？"——诱发"我妈/我婆婆就是这样"共鸣评论。
- **点赞 / 关注**：文末「赞👍🏻 + 在看❤️」+「关注温言入画」钩子 + #03 温言人设落点。
- **真实度**：刘姨事件据腾讯新闻公开报道，未添油加醋、未编造对话。
